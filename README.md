# ChatchMind Game
----------------------------------------
### 게임 목적
----------------------------------------
- 소켓 통신을 통해 서로 채팅을 하고 그림을 그리고 맞추는 게임을 자바 스윙으로 구현.

### 게임 실행 방법
----------------------------------------
1. IDServer 실행
 - 클라이언트의 접속을 대기하는 서버를 실행.
   
2. MainStart 실행
 - 서버에 입장하는 클라이언트 실행.
 - 다수로 접속 가능.
